# LASM: LLM Attack Surface Measurement Framework

A Unified Measurement Science Proposal for Security Evaluation of LLM-Based AI Systems.

This repository implements the LASM Framework outlined in `plan.md` (Version 1.1 + Game-Theoretic Extension), which provides:
- **LLM Attack Surface (LAS)** for attack surface quantification
- **Standardized ASR (SASR)** for standardized attack success rate measurement
- **Temporal Robustness Curve (TRC)** for temporal robustness evaluation
- **Economic Attack Surface (EAS)** capturing the economic feasibility of attacks
- **Attack Deterrence Threshold (ADT)** for determining optimal defense strength

Target Venues: IEEE TIFS, IEEE S&P, ACM TISSEC, Elsevier Computer Communications.
